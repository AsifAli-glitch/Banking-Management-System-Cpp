# Banking Management System (C++)

This project is a Banking Management System developed in C++ as part of the CodeAlpha Internship.  
It simulates basic banking operations and helps understand real-world implementation of file handling and structured programming.

## Features
- Create new bank accounts
- Deposit and withdraw money
- Check account balance
- View customer records
- File handling for data storage

## Tech Stack
- Language: C++
- Concepts Used: File Handling, Structures/Classes, Functions, Conditional Statements, Loops, Input/Output
